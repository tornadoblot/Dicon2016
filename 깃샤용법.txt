깃허브 매우 간단하게 쓰는방법
1. 깃허브 홈페이지 가서 회원가입을 한다
2. 오른쪽 밑에 New repository 라는 초록색 버튼을 누른다
3. Repository name을 정해주고 아무것도 건드리지 말고 Create repository를 누른다
4. 구글 창에서 git bash를 검색해 다운받아서 설치한다 그냥 yes하면 됨
5. 깃에다가 올리고 싶은 폴더로 가서 우클릭을 누르면 git bash here가 뜰테니 눌러라
5-1. 저거 안되면 너가 설치할때 잘못 건드린거임
6. 콘솔창이 하나 뜨면 홈페이지에 나와있는대로 치면 된다
7. git init
7-1. 이폴더가 이제부터 깃을쓸거라고 관리폴더를 넣어주는 명령어다
8. git add *
8-1. 원래는 git add 추가하고싶은 파일명.확장자까지 인데 *를 쓰면 추가안된 파일이 전부다 추가된다
9. git commit -m "어떤파일을 추가했는지 설명이나 코멘트" -m 꼭붙여라
10. git remote add origin https://github.com/니아이디/깃허브폴더이름.git
11. git push -u origin master
이러면 초기세팅 다한거고 나중에 파일 추가할때는
git add * -> git commit -m "asdf" -> git push origin master 만 하면 추가됨
특히 push 안하면 깃허브 홈페이지로 안올라가니까 푸시는 꼭 하고

다른애들꺼 깃 받는방법
git clone https://github.com/받고싶은애아이디/받고싶은깃폴더이름
하면 알아서 받아지고
클론 받아온 폴더는 업데이트를 너가 따로 해줘야함
친구껄 1일날에 클론 받았는데 2일날에 친구가 푸시를 해서 업데이트를 해줘야한다
그러면 git pull 하면됨
나머지는 라임이가 다 해줄꺼임
